# System Design социальной сети "Нам по пути"
System Design социальной сети "Нам по пути" для курса по System Design – https://balun.courses/courses/system_design

## Функциональные требования

- профиль путешественника (ник, ФИО, e-mail, телефон, фото, коротко о себе)
- профиль локаций (описание, страна, край/область/..., населенный пункт, гео-координаты)
- публикация постов о своих путешествиях
- - название
- - добавление новой локации или выбор существующей
- - даты посещения
- - фото с места
- - описание
- - собственная оценка (звезды от 0 до 5)
- поиск локаций для путешествий
- - рекомендации популярных мест по оценкам путешественников
- - поиск локаций по характеристикам (страна, город, ...)
- - просмотр ленты отзывов о локации
- - подписка на локацию
- поиск других путешественников
- - рекомендации путешественников
- - поиск путешественников по характеристикам
- - просмотр ленты путешественника
- - подписка на путешественника
- просмотр ленты путешественников, на которые подписан
- - оценить пост (звезды от 0 до 5)
- - оставить текстовый комментарий
- - подписаться на локацию путешествия
- просмотр ленты локаций, на которые подписан
- - оценить пост (звезды от 0 до 5)
- - оставить текстовый комментарий
- - подписаться на путешественика
- просмотр своей ленты путешествий

## Нефункциональные требования

- аудитория
- - DAU 10_000_000
- - только страны СНГ
- - сезонность не рассматриваем
- активность путешественников (в среднем)
- - 4 собственных поста в месяц
- - один раз в день читает ленту путешественников, на которых подписан
- - один раз в день читает ленту с локаций, на которые подписан
- - один раз в неделю рассматривает рекомендации по локациям 
- - один раз в неделю ищет новые локации
- - один раз в неделю рассматривает рекомендации по путешественникам
- - один раз в неделю ищет новых путешествеников
- лимиты на операции
- - максимальное число подписчиков 100_000
- - максимальное число подписок на путешественников 10_000
- - максимальное число подписок на локации 100
- - максимальное число просматриваемых постов в день - 50 
- - максимальное количество оценок и комментариев в день - 10 (по 5 на путешественников и локации)
- - макимальное количество рекомендаций в день - 10 локаций и 10 путешественников
